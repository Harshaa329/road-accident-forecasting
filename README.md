# 🚗 Road Accident Forecasting & Risk Analytics

Forecasting models built on 117,000+ road accident records to identify 
trends, risk patterns, and high-risk zones using SARIMA and LSTM.

## 🚀 Project Highlights
📊 **Dataset:** 117,000+ road accident records  
🧠 **Models:** SARIMA (statistical) + LSTM (deep learning)  
📍 **Clustering:** High-risk zone detection  
🎯 **Use Case:** Public safety analytics and risk prediction  

## 📂 Repository Structure
## 🧪 What I Built
- SARIMA and LSTM forecasting models on 117k+ records
- Clustering techniques to detect high-risk geographic zones
- Trend analysis and actionable safety insights

## 🛠️ Tech Stack
`Python` `SARIMA` `LSTM` `Clustering` `Time Series` `Pandas` `Matplotlib` `Seaborn`
